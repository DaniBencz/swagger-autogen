# Swagger-Autogen demo application

Prerequisites: `Node v18`

Install:
```
npm i
```

Start the server and generate `swagger_output.json` file:
```
npm start
```


Visit [localhost:3000/doc](http://localhost:3000/doc) for Swagger documentation
